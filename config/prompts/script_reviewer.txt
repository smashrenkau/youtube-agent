# 評価基準（各1〜5点）

## 1. 冒頭フック（hook_score）
- 5点: 最初の15秒で視聴者が絶対に見続けたくなる
- 3点: フックはあるが印象が弱い
- 1点: 冒頭から退屈で離脱リスクが高い

## 2. 話し方一致度（style_score）
- 5点: 自然な話し言葉で一貫したスタイル
- 3点: 部分的に書き言葉や硬い表現が混じる
- 1点: 読み上げ時に不自然な箇所が多い

## 3. 文字数適切さ（length_score）
- 5点: 目標文字数の±5%以内
- 3点: 目標文字数の±10%以内
- 1点: 目標文字数から±15%以上乖離

## 4. 構成の明確さ（structure_score）
- 5点: オープニング・本編・エンディングが明確で流れがスムーズ
- 3点: 構成はあるが一部つながりが弱い
- 1点: 構成が不明確で話が散漫

## 承認基準
合計16点以上 = 承認（approved: true）
15点以下 = 要修正（approved: false）

## 出力形式（必ずJSONで出力）
以下のJSON形式で出力してください。

{{
  "hook_score": 4,
  "style_score": 3,
  "length_score": 5,
  "structure_score": 4,
  "total": 16,
  "feedback": "改善点の詳細説明（修正時の具体的な指示を含める）",
  "approved": true
}}

JSONのみを出力し、それ以外のテキストは含めないでください。

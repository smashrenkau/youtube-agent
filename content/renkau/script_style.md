# 台本スタイルガイド

## 全体構成（5〜8分想定）
1. **つかみ（30秒）**：視聴者の悩みを言語化して共感を取る
2. **問題提起（1分）**：なぜそうなるのか原因を説明
3. **解決策の提示（2〜3分）**：具体的な方法を3つ前後で説明
4. **Renkauの紹介（1〜2分）**：自然な流れでサービスを紹介
5. **まとめ・行動促進（30秒）**：「まず知るだけでいい」で締める

## 文体ルール
- 話し言葉（〜です、〜ますより「〜だよね」「〜なんです」）
- 一文は短く（30文字以内を目安）
- 難しい言葉はすぐ解説を入れる
- 「あなた」に語りかける2人称を使う

## 禁止事項
- 不安を過度に煽る表現
- 「絶対」「必ず」などの断言
- 競合他社への直接的な批判
- 法的にグレーな表現

## 台本の長さ
- 目安：2500〜3500文字
- 読み上げ時間：5〜8分（1倍速）

## 構成例（タイトル：審査落ちでもスマホを持つ方法）
- つかみ：「スマホの審査に落ちたことありますか？」
- 問題：「審査落ちの原因は主に3つ」
- 解決策：「①中古を現金購入 ②格安SIM ③レンタル」
- Renkau紹介：「この中で一番リスクが低いのがレンタル、中でもRenkauは...」
- まとめ：「まずリンクを見るだけでOK」

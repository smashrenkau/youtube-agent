# 刺さるメッセージ・必須訴求ポイント

## Renkauとは
クレジットカード不要・審査なしで、新品の家電・スマホをレンタルできるサービス。
月額料金を払い続けることで、2年後に実質的に自分のものになる。

## 必ず台本に入れる4つのメッセージ

### 1. 総量規制に引っかからない・借入枠を温存できる
- レンタルは「借入」ではないため、総量規制の対象外
- カードローンやキャッシングの枠を使わずに済む
- 将来の信用再建を邪魔しない

### 2. 審査なし・クレジットカード不要
- クレジットカードなしでOK（デビットカード・銀行振込対応）
- 信用情報を照会しない（ブラックリスト・CIC登録関係なし）
- 収入証明・保証人も不要

### 3. 2年で実質的に自分のものになる（所有化）
- 月額を払い続けることで2年後に所有権が移転
- 「ずっと払い続けるだけ」ではなく、最終的に資産になる
- 途中解約も可能

### 4. 今すぐ生活を整えられる
- 冷蔵庫・洗濯機・エアコン・スマホなど生活必需品が対象
- 審査待ちなし、最短即日発送
- 生活の立て直しに必要なものをすぐ手に入れられる

## 差別化ポイント（競合との違い）
- 中古品ではなく新品
- ローン・借入ではないのでリスクゼロ
- 所有化できる（ただのレンタルとは違う）
